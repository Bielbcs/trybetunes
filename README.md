### Preview do projeto 💻



<details>
  <summary><h3>Como rodar localmente 👨‍💻</h3></summary></br>

1) Clone o repositório

```bash
$ git clone git@github.com:Bielbcs/trybetunes.git
```

2) Instale as dependências

```bash
$ npm install
```

3) Inicie o projeto

```bash
$ npm start
```
